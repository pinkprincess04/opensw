## LINUX 명령어 알아보자.
#### 20233062 김혜윤
<img src="https://github.com/user-attachments/assets/789ad783-969b-43fd-acb4-ca7f3a4c5986" alt="linux명령어알아보자" width="600px">

| 명령어 | 기능 | 주요 용도 | 예시 옵션 |
| :--- | :--- | :--- | :--- |
| **top** | 실시간 시스템 상태 및 프로세스 동적 모니터링 | 시스템 부하, CPU/메모리 사용률 확인 | P (CPU 정렬), M (Memory 정렬), k (프로세스 종료) |
| **ps** | 특정 시점의 프로세스 정적 스냅샷 확인 | 현재 실행 중인 모든 프로세스 목록 조회 | ps -ef, ps aux |
| **jobs** | 현재 셸에서 백그라운드/일시 정지된 작업 목록 관리 | 백그라운드 작업 상태 확인 및 포그라운드 전환 | fg %N, bg %N |
| **kill** | 특정 PID 또는 Job ID에 시그널을 보내 제어/종료 | 응답 없는 프로세스 강제 종료, 서비스 재시작 요청 | kill <PID>, kill -9 <PID> (강제 종료) |
